Add into Visual Studio Refference

»1) Now that the proxies have been generated, we can use them to access Java classes from .NET. Launch Visual Studio .NET and create a new Asp.net Web Site project

»2) Add to your project the file jnbauth_x86.dll or jnbauth_x64.dll or both (depending on whether the application will run as a 32-bit process, a 64-bit process, or either one).

» 3) Next Select The Added jnbauth_x86.dll or jnbauth_x64.dll file and go to the properties and Change Content Properties To (Content : Copy To All). When you done that the file is automaticaly added into your Refference after Build Project.
