Add Existing File In Project

» Add to your project the file jnbauth_x86.dll or jnbauth_x64.dll or both (depending on whether the application will run as a 32-bit process, a 64-bit process, or either one).

»If You Have An Error in For Adding The File Into Project I Recommended To Watch Video First.

»Next Select The Added jnbauth_x86.dll or jnbauth_x64.dll file and go to the properties and Change Content Properties To (Content : Copy To All). When you done that the file is automaticaly added into your Refference after Build Project.
