Add Web.config File

1) Add Web.config File in Your Project if exist already then nothing to do so.

2) Add the file app.config to the project. It is an application configuration file that configures the .NET side of JNBridgePro. You may want to examine the settings (it is set to communicate with the Java side using tcp/binary communications.

where the Java side is on the same machine as the .NET side and is listening on port 8085.

Make sure that, depending on whether you are using .NET Framework 2.0 or 4.0.

Please Remember you have commented and uncommented the appropriate sections of the configuration file. Depending on your System Platform x86 or x64 bit.
